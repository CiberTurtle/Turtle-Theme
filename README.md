# Turtle Theme

_v1.0.2_

A theme made by a turtle, for turtles.

Uses [Community Material Theme Darker High Contrast](https://marketplace.visualstudio.com/items?itemName=Equinusocio.vsc-community-material-theme) syntax colors.
